# LangChain Learning

LangChain 学习项目，基于 Python 3.12 + uv 管理。

## 项目结构

- `chapter_01` ~ `chapter_05`: 各章节学习笔记与代码（Jupyter notebook + Python 模块）
- `main.py`: 入口文件

## 技术栈

- 包管理: uv（uv.lock）
- 依赖: langchain, langchain-openai, langchain-community, langchain-deepseek, langsmith, jupyterlab
- Python: >=3.12

## 约定

- 使用约定式提交（Conventional Commits）
- API 密钥通过 `.env` 管理，不提交到仓库
